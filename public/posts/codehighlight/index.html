  • 代码块默认具有highlight的样式，可以调用。详情可以参考一下文档。 代码语法高亮chroma设置的文档链接 github库

以下是使用方法： ```go {linenos=false hl_lines=[3,“6-8”] style=emacs} 来调整` … some code here…

{中的参数含义} linenos=false – 没有行号 style=emacs – 样式

  • 没有行号的显示方式
package main

import "fmt"

func main() {
    for i := 0; i < 3; i++ {
        fmt.Println("Value of i:", i)
    }
}
  • 短代码实现macos样式的代码块 下图是一个使用md中使用shortcode的例题：

短代码模版要定义到shortcodes目录下 layouts/shortcodes/terminal.html

 1{{ $lang := .Get "lang" | default "text" }}
 2{{ $title := .Get "title" | default "Terminal" }}
 3
 4<div class="terminal">
 5  <div class="terminal-header">
 6    <span class="terminal-dot red"></span>
 7    <span class="terminal-dot yellow"></span>
 8    <span class="terminal-dot green"></span>
 9    <span class="terminal-title">{{ $lang }}</span>
10  </div>
11  <div class="terminal-body">
12    {{ highlight .Inner $lang "linenos=inline" }}
13  </div>
14</div>
